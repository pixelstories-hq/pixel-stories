title: Editor Overview
description: Learn the main areas of the PS Maker editor.
---


When you first open a project in PS Maker, you start in the map editor. This is the main workspace where you build the world your players will explore.

Below is generally the main areas you'll use

## Top navigation

In the top left, there are tabs for the map editor, project assets, and play testing. The map editor is where you build maps, Project assets is where you manage tilesets, players, NPCs, and other game assets, and Play test lets you play through your game.


## Map dropdown

Next to those tabs is the map dropdown. Use this to navigate between the maps you have created and open the one you want to work on.

## Map and Events Modes

In the top left, there are also two modes for the editor, map and events. Use Map to edit tiles, objects, and collisions. Use Events to configure game logic like interactions, dialogue, spawning NPCs, moving characters, and changing maps.

## Map editing toolbar 

When you're in the map editing mode, there is a toolbar at the bottom center of the map. Use these tools to draw tiles, place objects, and add collisions. Try each one for a bit to get a feel for how the editor wants to be used.

## Left panel

You'll find a panel on the top left that will show based what you've selected in the editor, such as a tileset for draw tool, map objects when placing objects, collisions, or an event's action list.
