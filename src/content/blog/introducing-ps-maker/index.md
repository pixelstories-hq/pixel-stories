## It’s Finally Released!

Hey everyone! The long awaited [PS Maker App](https://tdgao.itch.io/ps-maker) is finally out! I've released it both on the Pixel Stories website and on itch.io. You can download the demo for free and try all the features with it. The app is a continuation of the previously made PS Maker web-app (which was a somewhat poor design decision to be in a website, app works much better) and it makes many, many improvements on it. I've fleshed out the entire project asset system and events/actions system. To prove that everything works, I recreated [Dead Friend Horror](https://tdgao.itch.io/dead-friend-horror), the example story game project made with PS Maker. You can find that game in my itch page as well.

Hope you guys enjoy the release and let me know if you have suggestions for features or encounter any bugs! Feel free to use the [Github issues tracker](https://github.com/pixelstories-hq/ps-maker/issues/new/choose) for that.
