## Description

- A tileset is an image made of smaller tiles that you paint onto map layers.
- An autotile is a special terrain image that automatically chooses edges, corners, and fills as you draw.
- Use regular tiles for base terrain and details, then use autotiles for shaped patches like paths, water, cliffs, or carpets.
