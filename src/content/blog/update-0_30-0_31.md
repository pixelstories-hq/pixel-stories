Hey everyone!! I want to give a small update on the recent updates for PS Maker, 0.30 and 0.31! These two updates added two major things for PS Maker.

0.30 adds scoping for events, meaning now event triggers can own events and get duplicated when copied, and fully deleted when deleted. Map events still stay as reusable events any trigger can use. Global events are also added where an event can be used in multiple maps. It also added an ability to spawn in NPCs from the map editor tab using a spawn NPC tool.

0.31 adds an improved editor with selection tools and full move/delete and copy/paste functionality, while keeping undo/redo integrated with it. I also reorganized the toolbar to make more sense for how things are laid out now with adding map objects, npcs/collision and tile editing.

There are also many smaller things that were added inbetween these updates! Some highlights from the changelog are below. You can find the full changelog in the changelog page :). Now, I'm going to do a round of smaller misc things that have been requested but don't belong with any major update. Stay tuned!

## 0.31 key changes

- Added a tile selection tool with drag to move and copy/paste support.
- Added a selection tool for map objects, NPCs, and collisions with copy/paste and multi-select to move.
- Added a universal delete object tool and reorganized the toolbar, including new toolbar shortcuts and context menus.
- Added a draw event trigger tool and an event toolbar with a dedicated select event tool.
- Made event triggers persistent, selectable areas on the map: move, resize, double click to open the action config, and select/move/delete multiple at once.
- Changed the event editor preview to show the map state up to the selected action.

## 0.30 key changes

- Added global, map-scoped, and trigger-scoped events with event ownership, so copy/pasted actions deep copy the events they own.
- Redesigned the NPC creation workflow: auto-generate animations from spritesheets and bulk import to create multiple NPCs at once.
- Added map setup tools in the map tab for spawning NPCs and moving NPCs or the player.
- Improved action drag and drop with smoother group dragging and moving actions in and out of conditional branches.
- Added a Run Event action.
- Fixed the "Framebuffer unsupported" crash, a memory leak from deferred WebGL canvas cleanup.
