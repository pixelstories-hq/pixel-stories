PS Maker is a game engine for creating 2D, top-down, story-focused games. These guides help you find your way around the editor and learn as you build.

Have fun creating with PS Maker!

## Getting started

If you are new, these are the pages worth reading first:

- Start with [Editor Overview](/introduction/editor-overview/) to get familiar with the interface.
- Then read [Events and Actions](/event-system/events-actions/), which are the heart of how things happen in your game.
- After that, learn about [Triggering Events](/event-system/event-triggers/) so your world can react to the player.

After that, jump around as you go! The help manual is made to be wandered through, not read front to back (otherwise its way too much information).

If you're stuck and the manual doesn't help, [join the Discord](https://discord.gg/WTxUC4hEnS) to ask for help. You can ping me, @Truman, and I'll try to respond as soon as I can!
