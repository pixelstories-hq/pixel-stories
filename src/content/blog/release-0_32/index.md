Hello everyone! I know there's been some requests to add more assets for PS Maker. So, in update 0.32, I introduce... Tiles 'N Such assets!

This asset pack has more NPCs, more autotiles, a bunch of map objects for both interior and exterior environments. Theres also fences and bridges!

It also comes with a showcase project that you can check out the assets in.

A big thank you to Signus, who put together these assets. Also, check out this video Signus made to showcase the new assets. At the end theres also a short Q&A!

Besides the assets, I also fixed up some bugs and improved performance for projects with a lot of NPCs.

Hope you enjoy the update!

## 0.32.0

- Added tiles n such starter asset pack!
- Added explicit game link generation and updating.
- Added click to set player sort position and drag to draw a collision box.
- Added next/prev buttons to view a lot of assets.
- Improved performance with asset resolver cache for filesystem images and animation previews.
- Changed collisions and player sort point to always show for editing.
- Changed the boolean input in conditionals to a dropdown.
- Persisted map editor panel scroll.
- Kept scroll position in project assets content.
- Fixed player config not opening when no animations are set for it.

See the full changelog for [PS Maker v0.32.0](/changelog#0320).
