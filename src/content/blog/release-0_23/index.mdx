Sooo PS Maker really ain't much without assets. To fix that, today I’m releasing PS Maker Starter Assets!

This updates comes with 2 new things:

- New starter asset packs that comes integrated with PS Maker
- A web-based asset pack maker made for creating PS Maker assets

PS Maker now comes with 4 starter asset packs you can add to your project. Unlike typical assets, these assets come with the metadata which makes it instantly ready to use in the engine, so you don’t have to go through importing everything, adding collisions, setting up autotiles, NPC animations, etc etc.

[video clicking on browse assets, then opening an asset pack, scroll down, then add one asset, then add all assets]

To create the PS Maker asset packs in an efficient way, I had to create the PS Asset Pack Maker. This tool essentially takes the asset creation flows from the engine and put them into a separate web-based tool anyone can use. The exported pack is a `.pspack` file, which can then be imported into the engine to use as well.

[video of calciumprice map objects, then opening create map object, add collision, and click create]

I sourced these assets from [OpenGameArt.org](http://OpenGameArt.org), so they are free to use in your games with attribution. Combined, there are over 200 assets to choose from! Huge thanks to these artists and creators who made the asset packs!
