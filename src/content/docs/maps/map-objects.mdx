title: Map Objects & Collisions
description: Place objects on maps, use automatic depth sorting, and block player movement with collisions.
---

## Description

- Map objects do not live on tile layers; they use their own object layering system.
- Objects are automatically depth sorted, so the player can draw in front of or behind them based on position.
- Collisions are separate invisible areas that decide where the player can and cannot walk.
