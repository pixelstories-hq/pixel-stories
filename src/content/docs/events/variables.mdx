## Description

- Use switch variables for true-or-false state like `hasKey` or `metGhost`.
- Use number variables for counts, scores, puzzle values, and progress.
- Use text variables for named states or short pieces of remembered text.
- Use conditionals to branch dialogue, unlock paths, check inventory, and create multiple outcomes.
